PACMAN CHEATSHEET

# INSTALL PACKAGES
Install Package: pacman -S <package>
Install Package And Update Package List: pacman -Syu <package>
Install Package From Local Location: pacman -U /path/to/local/<package>
Reinstall All Repo-Based Packages (During Emergencies): pacman -Qnq | pacman -S -

# REMOVE PACKAGES
Remove Package: pacman -R <package>
Remove Package And Dependencies Not Tied To Other Packages: pacman -Rs <package>
Remove Package, Dependencies And Configuration: pacman -Rns <package>

# UPDATE PACKAGES / SYSTEM
Update System: pacman -Syu
Only Update Software Database: pacman -Syy

# QUERY PACKAGES
List Installed Packages: pacman -Qn
Search For Installed Packages: pacman -Qs <package>
Search For Packages: pacman -Ss <package>
List Information About A Package: pacman -Qii <package>
List Files Included In A Package: pacman -Ql <package>
List Packages No Longer Required By Others: pacman -Qdtq

# ORPHAN PACKAGES
List Unneeded Packages: pacman -Qdt
Uninstall Unneeded Packages: pacman -Rns $(pacman -Qdtq)

# DEPENDENCIES
What Does A Package Depend On: pactree <package>
What Depends On A Package: pactree -r <package>

# CACHE
Clean Old Packages In Cache: pacman -Sc
